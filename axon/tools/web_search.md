Search the web and return cleaned content from top results. Each result is numbered [1], [2], etc.

Inputs:
- query (required): The search query string.
- allowed_domains (optional): Only include results from these domains.
- blocked_domains (optional): Exclude results from these domains.
- max_results (optional): Maximum number of results to return (default: 5, max: 10).

Notes:
- When citing information from search results in your response, use the result numbers as inline citations like [1], [2], or [1,2].
