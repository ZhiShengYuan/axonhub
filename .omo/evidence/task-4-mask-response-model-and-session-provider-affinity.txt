- Store key is scope + "\x00" + sessionID.
- Provider identity is string(candidate.Channel.Type) via ProviderKey.
- Store is bounded with default max entries 1024 and evicts oldest inserted entry when exceeded.
- Empty scope/sessionID are ignored and return !ok.
- WithScopedSessionID reads shared.GetSessionScope(ctx) and shared.GetSessionID(ctx).
